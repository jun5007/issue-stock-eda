# 이슈와 주식 EDA: 개인투자자의 FOMO 행동 패턴 분석

> 뉴스 건수와 검색량으로 시장의 이슈강도를 정의하고, 종목별 주가/거래량 반응과 개인투자자 FOMO 행동 패턴을 탐색한 프로젝트입니다.

## Overview

- 진행 기간: 2025.09 - 2025.12
- 학년/학기: 2학년 2학기
- 소속: 순천향대학교 AI빅데이터학과
- 형태: 탐색적 데이터 분석 팀 프로젝트
- 팀원: 홍찬기, 추민주, 엄용현, 이석준
- GitHub Portfolio: [jun5007.github.io](https://jun5007.github.io)

## Problem

특정 기업 이슈가 발생하면 뉴스 노출과 검색량이 급증하고, 개인투자자의 매수/매도 행동도 함께 변할 수 있습니다. 이 프로젝트는 단순히 "이슈가 많으면 주가가 오른다"는 가설을 넘어서, 이슈가 가격 방향성보다 거래량과 투자자 관심도에 어떤 방식으로 연결되는지 데이터로 확인했습니다.

핵심 질문은 다음과 같습니다.

- 뉴스와 검색량 spike가 실제 시장 반응으로 이어지는가?
- 이슈강도는 다음 거래일 수익률을 설명할 수 있는가?
- 종목별로 이슈에 민감하게 반응하는 유형이 존재하는가?

## Data

- KRX 주식 정보:
  - 일별 시가, 고가, 저가, 종가
  - 거래량, 거래대금, 시가총액
  - 개인 매수/매도/순매수
  - 외국인 보유수량, 외국인 지분율
- 네이버 데이터랩:
  - 종목별 검색량
- 빅카인즈:
  - 종목별 뉴스 기사 건수
  - 주요 이슈 기사
- 분석 대상 예시:
  - 삼성전자, NAVER, 카카오, 현대차, LG에너지솔루션, 셀트리온, 에코프로, 펄어비스, 포스코퓨처엠, 한화오션
- 최종 통합 데이터:
  - 3,620행 x 29개 변수

## Preprocessing

- 종목별 Wide Format 데이터를 `date`, `종목`, `변수` 기준 Long Format으로 변환했습니다.
- 뉴스/검색 데이터와 주식 데이터를 `date`, `종목` 기준으로 병합했습니다.
- 휴장일 이슈를 반영하기 위해 다음 거래일(`next_trade_date`)을 매핑했습니다.
- 가격/보유량 계열은 Forward Fill로 보정하고, 거래량/수급 같은 흐름 변수는 결측 시 0으로 처리했습니다.
- 뉴스 건수와 검색량의 편차를 줄이기 위해 로그 변환을 적용했습니다.

## Feature Engineering

| Feature | Description |
|---|---|
| `뉴스_log` | `log1p(뉴스건수)` |
| `검색_log` | `log1p(검색량)` |
| `이슈강도` | `뉴스_log + 검색_log` |
| `등락률_abs` | 등락률 절댓값 |
| `거래량_log` | `log1p(거래량)` |
| `등락률_next` | 다음 거래일 등락률 |
| `거래량_log_next` | 다음 거래일 로그 거래량 |

## Analysis

### Issue Sensitivity Clustering

종목별 수익률 변동성과 거래량 스파이크를 표준화한 뒤 K-Means 클러스터링을 수행했습니다. Silhouette Score만 보면 `k=2`가 가장 높았지만, 이슈 반응 강도 차이를 더 잘 설명하기 위해 `k=3`을 선택했습니다.

| Cluster | Type | Stocks | Interpretation |
|---|---|---|---|
| 0 | 고민감형 | 펄어비스, 셀트리온 | 이슈 발생 시 수익률 변동성과 거래량 반응이 크게 나타남 |
| 1 | 저민감형 | 카카오, 현대차, 삼성전자, 한화오션, NAVER, 포스코퓨처엠, LG에너지솔루션 | 관심도와 거래량은 높지만 가격 반응은 비교적 제한적 |
| 2 | 중간민감형 | 에코프로 | 기본 변동성이 높아 추가 이슈에 대한 거래량 민감도는 제한적 |

### Event Study

종목별 이슈강도 Top1 날짜를 기준으로 이벤트 전후 주가와 거래량 반응을 비교했습니다.

- 셀트리온: 관세 리스크 해소 관련 긍정 이슈 이후 당일 수익률과 거래량이 크게 반응했습니다.
- 펄어비스: 실적 부진과 출시 연기 악재 이후 급락과 높은 거래량이 함께 나타났습니다.
- NAVER: 합병 기대와 해킹 악재가 동시에 노출되며 가격 방향성이 단순하지 않았습니다.

### Correlation Analysis

- 이슈강도와 다음 거래일 등락률의 전체 상관계수는 약 `0.03`으로 매우 낮았습니다.
- 이슈강도와 다음 거래일 거래량의 상관계수는 약 `0.56`으로 더 뚜렷했습니다.
- 전 종목에서 이슈강도와 다음날 거래량 변화의 상관계수가 양수로 나타났습니다.

## Result

이슈강도는 다음날 주가 방향을 직접 예측하는 변수라기보다, 투자자 관심도와 거래 참여 강도를 설명하는 지표에 더 가깝다는 결론을 얻었습니다.

뉴스와 검색량이 커질수록 다음날 거래량은 증가하는 경향이 나타났지만, 수익률은 종목별 이슈의 성격, 투자 심리, 기업 고유 이벤트에 따라 다르게 반응했습니다.

## Limitations & Next Steps
